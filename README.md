# JARVIS-C Phishing Detection System

JARVIS-C is an AI-powered cybersecurity system that detects phishing websites in real time using machine learning and browser security analysis.

## Features
• Machine Learning phishing detection  
• Chrome browser extension  
• Real-time URL scanning  
• SSL certificate validation  
• Domain age verification (WHOIS)  
• Safe / Phishing classification  

## Project Architecture

User Browser
↓
Chrome Extension
↓
Backend API (Flask)
↓
Machine Learning Model
↓
Security Feature Extraction
↓
Phishing Detection Result

## Technologies Used
Python  
Flask  
Scikit-learn

JavaScript  
Chrome Extension API  
WHOIS Lookup  

## Installation

Clone repository

git clone https://github.com/Shashankkarnati12/jarvis-c-phishing-detector

Install dependencies

pip install -r requirements.txt

Run API server

python src/api_server.py

Load Chrome Extension

1. Open chrome://extensions  
2. Enable Developer Mode  
3. Click Load Unpacked  
4. Select extension folder
